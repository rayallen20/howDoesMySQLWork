# 2. 常量传递

有时某个表达式是某个列和某个常量做等值匹配,例如:

```
a = 5
```

当使用AND操作符将该表达式和其他涉及列`a`的表达式连接时,可以将其他表达式中`a`的值替换为5,例如:

```
a = 5 AND b > a
```

就可以被转换为:

```
a = 5 AND b > 5
```

注意: 如果是OR操作符,则不能进行这样的转换,例如:

```
a = 5 OR b > a
```

若改写为:

```
a = 5 OR b > 5
```

则`a = 3, b = 4`的这条数据在改写后就不满足条件了.本质原因是因为OR操作符是在扩大结果集的范围,而AND操作符是在缩小结果集的范围.
因此,在使用OR操作符时,不能将常量替换为列的值.
