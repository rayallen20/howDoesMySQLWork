# 2. 相关子查询

- 相关子查询: 若子查询的执行需要依赖于外层查询的值,则该子查询称为相关子查询

例如:

```
mysql> SELECT *
    -> FROM t1
    -> WHERE m1 IN (
    ->     SELECT m2
    ->     FROM t2
    ->     WHERE t2.n2 = t1.n1
    -> );
+------+------+
| m1   | n1   |
+------+------+
|    2 | b    |
|    3 | c    |
+------+------+
2 rows in set (0.00 sec)
```

本例中子查询为`(SELECT m2 FROM t2 WHERE t2.n2 = t1.n1)`.该查询中有一个搜索条件是`t2.n2 = t1.n1`.
`n1`是表`t1`的列,也就是外层查询的列,也就是说子查询的执行需要依赖于外层查询的值,所以该子查询为相关子查询.
