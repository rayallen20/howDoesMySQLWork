# 1. 标量子查询

- 标量子查询: 只返回**1个单一值**的子查询
  - 1个单一值又叫做**标量**

例如: 

```
mysql> SELECT (
    ->     SELECT m1
    ->     FROM t1
    ->     LIMIT 1
    -> );
+-------------------------------------------+
| (
    SELECT m1
    FROM t1
    LIMIT 1
) |
+-------------------------------------------+
|                                         1 |
+-------------------------------------------+
1 row in set (0.00 sec)
```

或:

```
mysql> SELECT *
    -> FROM t1 
    -> WHERE m1 = (
    ->     SELECT MIN(m2)
    ->     FROM t2
    -> );
+------+------+
| m1   | n1   |
+------+------+
|    2 | b    |
+------+------+
1 row in set (0.02 sec)
```

这2个查询语句中的子查询都返回1个单一的值(也就是1个标量).这些标量子查询可以作为**1个单一值或表达式的一部分**出现在查询语句的各个地方
