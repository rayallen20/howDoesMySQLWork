# 4. 表子查询

- 表子查询: 查询既包含很多条记录,又包含很多个列的子查询

例如:

```
mysql> SELECT *
    -> FROM t1
    -> WHERE (m1, n1) IN (
    ->     SELECT m2, n2
    ->     FROM t2
    -> );
+------+------+
| m1   | n1   |
+------+------+
|    2 | b    |
|    3 | c    |
+------+------+
2 rows in set (0.00 sec)
```

其中`(SELECT m2, n2 FROM t2)`就是表子查询.这里需要和行子查询对比一下:

- 行子查询中使用了`LIMIT 1`来确保子查询的结果只有1条记录
- 表子查询则不需要这个限制
