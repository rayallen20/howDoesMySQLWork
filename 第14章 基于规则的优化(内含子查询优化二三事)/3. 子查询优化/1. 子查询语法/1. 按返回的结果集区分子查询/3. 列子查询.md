# 3. 列子查询

- 列子查询: 返回1个列的数据的子查询
  - 不过这个列的数据需要包含多条记录(只包含1条记录就成了标量子查询了)

例如:

```
mysql> SELECT *
    -> FROM t1
    -> WHERE m1 IN (
    ->     SELECT m2
    ->     FROM t2
    -> );
+------+------+
| m1   | n1   |
+------+------+
|    2 | b    |
|    3 | c    |
+------+------+
2 rows in set (0.00 sec)
```

其中,`(SELECT m2 FROM t2)`就是列子查询.表明查询出`t2`表的`m2`列的值作为外层查询IN语句的参数
