# 1. IN或者NOT IN

`IN`/`NOT IN`与子查询组成的布尔表达式的格式如下:

```
操作数 [NOT] IN (子查询)
```

该布尔表达式的含义为: 用来判断某个操作数是否存在于由子查询结果集组成的集合中.

例如:

```
mysql> SELECT *
    -> FROM t1
    -> WHERE (m1, n1) IN (
    ->     SELECT m2, n2
    ->     FROM t2
    -> );
+------+------+
| m1   | n1   |
+------+------+
|    2 | b    |
|    3 | c    |
+------+------+
2 rows in set (0.00 sec)
```

该查询含义为: 找出`t1`表中的某些记录,这些记录的`m1`和`n1`列存在于子查询的结果集中.
