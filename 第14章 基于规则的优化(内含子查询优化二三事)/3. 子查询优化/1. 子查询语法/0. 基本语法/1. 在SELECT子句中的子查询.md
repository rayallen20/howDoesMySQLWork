# 1. 在SELECT子句中的子查询

例如:

```
mysql> SELECT (
    ->     SELECT m1
    ->     FROM t1
    ->     LIMIT 1
    -> );
+-------------------------------------------+
| (
    SELECT m1
    FROM t1
    LIMIT 1
) |
+-------------------------------------------+
|                                         1 |
+-------------------------------------------+
1 row in set (0.00 sec)
```
